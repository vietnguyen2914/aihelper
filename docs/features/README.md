# Features

## Purpose

Feature docs group related use cases into a business capability cluster.

## What To Record

- Feature name
- Business goal
- Important entry points
- Core entities
- Related use cases
- Overlaps with other features
- Notes about constraints or edge cases

## Template

```text
# <Feature Name>

## Purpose

## Entry Points

## Core Entities

## Related Use Cases

## Overlaps

## Notes
```
