# Testimonials

Use this page to collect real user feedback before adding quotes to the README.
Do not invent testimonials.

## What To Collect

Ask users for:

- workflow before aihelper,
- workflow after aihelper,
- repo size or context size if they can share it,
- latency/token reduction if measurable,
- editor used,
- permission to quote publicly.

## Request Template

```markdown
Thanks for trying aihelper. If it saved you time or reduced token usage, could
you share a short public quote?

- What workflow did it improve?
- What editor/model setup did you use?
- Can I quote your name/GitHub handle?
```

## README Criteria

Only add a quote to README when it is:

- from a real user,
- specific about workflow value,
- approved for public use,
- not just generic praise.
