# v0.0.6 Release Notes

## Headline

Stop sending giant prompts: aihelper routes coding agents to compact, relevant
context through a local semantic runtime.

## Highlights

- Demo-first README with wider workflow GIF previews.
- Regenerated diagnostics and OCR GIF demos at a shorter, more readable size.
- Architecture visual, latency chart, token reduction chart, and memory tier chart.
- 30-second bootstrap plus `aihelper doctor`.
- Semantic routing for compact context instead of broad repo scans.
- Daemon runtime for low-latency local workflows.
- Diagnostics -> patch planning -> confidence scoring -> safe apply workflow.
- Screenshot/OCR parsing workflow for multimodal debugging.
- Cross-editor MCP setup for Zed, Claude Desktop, Gemini, Codex, OpenCode, and VSCode.
- Contributor guide, changelog, issue templates, PR template, and funding metadata.

## Try It

```bash
git clone https://github.com/vietnguyen2914/aihelper.git
cd aihelper
bash scripts/bootstrap.sh
./bin/aihelper doctor
```

Then run it in a project:

```bash
aihelper cache build
aihelper route "fix payment bug"
```
