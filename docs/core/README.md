# Core Project Map

## Purpose

This page captures the base project structure that sits underneath the business docs.

## What Belongs Here

- Bootstrapping
- Local environment setup
- Runtime entrypoints
- Shared configuration
- Project conventions
- Test and verification paths

## Notes

- Keep this page framework-neutral.
- If the project is PHP, document the application entrypoints, shared libraries, and local bootstrap scripts here.
- If the project is Java, document startup, build, and generated scaffolding here.
